Reverse complements are useful in various molecular biology applications, such as creating primers for PCR. For more information about complements refer to this. Reverse complements are easily done by using perl's reverse and transliteration functions (tr//) with some regular expression experience. This example utilizes Fast CGI as a web platform with apache.
