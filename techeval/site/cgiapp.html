CGI::Application is an easy to use web application tool kit. It is easy to use and has good integration with other core technologies commonly used in web applications.

You may find these slides with notes a useful reference.

This app is a simple book database. It uses the same database and templates as the catalyst example but does not implement a log in system. Anyone can add or delete books to the database.

This demonstrates basic "Create Retrieve Update Delete" database operations using CGI::Application as the backbone and Template Toolkit as the templating engine.

The source code for the application

The main modules are Books.pm the CGI::Application subclass and BookModel.pm. They can be found in the lib directory.
